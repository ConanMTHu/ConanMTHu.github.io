Facebook POP使用介绍

Animation
  • pop

契机

项目最近加了个需求,统计所得收益的金额(LZ做的app金融属性很强)要像余额宝一样带跳动效果.网上找到篇文章可以实现这里,不过感觉很麻烦,再查了一下说POP可以实现,顺便来介绍下,虽然已经出来很久了…so->上Demo

POPBasicAnimation
POPSpringAnimation
POPDecayAnimation
POPCustomAnimation

let moveYAnimation = POPBasicAnimation(propertyNamed: kPOPLayerPositionY)
moveYAnimation.toValue = 500
moveYAnimation.duration = 3.0
// 先慢->快->再慢
moveYAnimation.timingFunction = CAMediaTimingFunction(name: kCAMediaTimingFunctionEaseInEaseOut)
numberLabel.pop_addAnimation(moveYAnimation, forKey: "MovePositionY")

// Common CALayer property names.

extern NSString const kPOPLayerBackgroundColor;
extern NSString const kPOPLayerBounds;
extern NSString const kPOPLayerCornerRadius;
extern NSString const kPOPLayerBorderWidth;
extern NSString const kPOPLayerBorderColor;
extern NSString const kPOPLayerOpacity;
extern NSString const kPOPLayerPosition;
extern NSString const kPOPLayerPositionX;
extern NSString * const kPOPLayerPositionY;
//………..太多了,不一一列举 ……………//

CGFloat springBounciness;// 弹簧弹力,值越大震动幅度越大 范围为[0, 20]，默认值为4
CGFloat springSpeed;// 弹簧速度,速度越快动画时间越短 范围为[0, 20]，默认为12
// 上面两个属性一起决定着弹簧动画的效果
CGFloat dynamicsTension;// 弹簧的张力
CGFloat dynamicsFriction;// 弹簧的摩擦力
CGFloat dynamicsMass;// 弹簧的质量

let springAnimation = POPSpringAnimation(propertyNamed: kPOPLayerPositionY)
springAnimation.toValue = 500
springAnimation.springBounciness = 16
springAnimation.springSpeed = 1
numberLabel.pop_addAnimation(springAnimation, forKey: "springAnimation")

velocity // 速率,必须和你操作的属性有相同的结构
deceleration // 衰减系数(负加速度),越小则衰减得越快,默认是0.998,一般不去修改

有种刹车的感觉
let decayAnimation = POPDecayAnimation(propertyNamed: kPOPLayerPositionY)
decayAnimation.velocity = 3600
decayAnimation.deceleration = 0.998
numberLabel.pop_addAnimation(decayAnimation, forKey: "decayAnimation")

readBlock // 当前的属性值
writeBlock // 修改变化后的属性值
threashold // 值越大writeBlock的调用次数越少,默认不去设定
`` obj就是我们的Label,values这个是动画作用的属性数组,这里我们只需要操作Label上显示的文字，所以只需要一个参数为values[0],如果是多个比如CGRectMake(x,x,x,x),则是values[0] values[1] values[2] values[3]了. 是不是非常的简单!PNChat里面使用的是一个UICountingLabel`的第三方,也能实现这个效果.
更多的自定义动画就等大家去探讨了,LZ工作过程中有新的动画也会继续加进来
